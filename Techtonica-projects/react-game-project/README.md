## Week 5 Project: React Memory Game
**Introduction: <br>**
Wild Pairs is a fun memory card game where players are challenged to find pair up cards with matching animal images. The goal is to flip 2 cards at a time and match all cards displayed. This project was created for Week 5's projects for Techtonica program. Wild Pairs uses React, HTML and CSS.

**How to Play:**
1. Flip over two cards at a time by clicking on them.
2. Score points for each pair of matching cards found.
3. The game ends when all pairs have been found.
4. Click the "New Game" button to start a new round.

**Technologies: <br>**
- HTML: Used to structure game layout and content.<br>
- CSS: Applied styling to the game interface, creating better User Experience and User Interface.<br>
- JavaScript: Used to implement game logic - shuffling cards and comparing selected cards.<br>
- React: Used to set and manage states based on user interaction - user turns, selecting cards and revealing matches.<br>

**Installation: <br>**
To set up Wild Pairs on your local machine, follow these steps: <br>
- Clone repository onto your local machine: <code>git clone https://github.com/stmcpeters/Techtonica-Assignments.git</code><br>
- Navigate to the react game project directory: <code>cd techtonica-assignments/techtonica-projects/react-game-project</code><br>
- Install NPM packages using: <code>npm install</code><br>
- Run game using: <code>npm start</code><br>
- Have fun!<br>
